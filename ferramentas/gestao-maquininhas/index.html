Importar transações

▸

Envie o arquivo exportado da Dock (.xlsx ou .csv) — pode ser o relatório de um mês ou o histórico completo desde o início: o sistema não duplica transações já importadas. O sistema soma o valor transacionado aprovado por CNPJ, identifica a marca (Oncoprod/Farma), atualiza os clientes já cadastrados e cria automaticamente os clientes novos que aparecerem no arquivo — já com a data de entrega do arquivo preenchida, faltando só a data de retirada.

Importar Ordens de Serviço (TEFTI)

▸

Envie o relatório de Ordem de Serviço exportado da TEFTI (.xlsx ou .csv). O sistema lê OS de instalação, desinstalação, manutenção e troca de tecnologia por CNPJ, atualiza a lista de maquininhas de cada cliente (modelo, data de entrega, retirada, rastreio) e guarda todo o histórico de OS. Clientes novos que aparecerem no arquivo são criados automaticamente. OS de Manutenção e Envio de Bobinas entram só no histórico e não alteram a lista de maquininhas.

Importar Estabelecimento

▸

Envie o arquivo de cadastro de estabelecimentos (.xlsx ou .csv), com colunas como Razão Social, CNPJ-CPF, E-mail, Telefone, Cidade, Estado e Tabela de Preços. O sistema atualiza esses dados no perfil de cada cliente pelo CNPJ. Se o CNPJ ainda não existir na base, um cadastro novo é criado (sinalizado como incompleto) para depois ser completado com maquininhas e transações.

Configuração de modelos de terminal

▸

Valores padrão de mensalidade e mínimo mensal para isenção, por modelo. Servem de base para o cálculo de cobrança de todos os clientes — para negociar um valor diferente com um cliente específico, use o campo "Valor de mensalidade" na seção Maquininhas do Perfil dele.

Como o cálculo funciona: mínimo exigido e mensalidade são somados por terminal (quantidade × valor do modelo). Dentro dos primeiros 12 meses da Data de Entrega, o cliente fica isento no mês se o volume transacionado atingir esse mínimo; depois dos 12 meses, a mensalidade é cobrada sempre. Um bloqueio manual ativo suspende qualquer cobrança enquanto durar. O modelo S920 foi incluído usando a mesma regra do MOVE5000/PPC930 (R$ 47 / mínimo R$ 15.000) — ajuste se a regra for diferente. Datas de transação são interpretadas no formato do arquivo de origem (dia/mês/ano), sem inversão.
